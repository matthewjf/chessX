# Chess X
## Local setup

1. Install and run MongoDB ([instructions](https://docs.mongodb.com/manual/tutorial/install-mongodb-on-os-x/))
- Run `$ npm install`
- Run `$ bower install`
- Run `$ npm run watch`
- In a separate tab, run `$ gulp`
