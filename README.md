# Chess X
## Local setup

1. Install MongoDB & run ([instructions](https://docs.mongodb.com/manual/tutorial/install-mongodb-on-os-x/))
3. Install & run redis (`$ redis-server &`)
- `$ npm install`
- `$ npm run watch`
- In a separate tab, run `$ gulp`
